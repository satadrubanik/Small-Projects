# PromptForge Showcase

A small portfolio repo containing two AI projects built with OpenAI:

- `Netflix Campaign Generator`
- `Nestle HR Policy Assistant`

## Contents

- Text-to-image campaign generator for Netflix-style promotional assets
- RAG-based HR policy assistant for querying policy documents

## Notes

- The notebooks are kept in separate folders with descriptive project names.
- The HR assistant folder includes the source PDF used for retrieval.
- Generated vector stores and temporary notebook checkpoints are intentionally excluded.
